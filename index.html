Malaysia Weather Dashboard

This dashboard provides an overview of Malaysia's weather patterns using interactive charts and maps. Explore precipitation, humidity, wind speed, UV index, and more across different months and regions.

Hourly Temperature by Malaysian State

This map shows hourly temperature across Malaysian states. Use the dropdown menus to filter by year, month, day, and hour.

You can zoom and re-center the map using the controls. The color scale indicates temperature intensity from cooler to hotter.
