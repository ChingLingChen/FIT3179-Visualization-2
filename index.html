Malaysia Weather Dashboard

Welcome to the Malaysia Weather Dashboard for 2025! This interactive tool visualizes weather data across Malaysian states, including temperature, precipitation, humidity, wind speed, UV index, and more. Use the maps, charts, and filters to explore seasonal trends and plan accordingly for agriculture, travel, or daily activities. Data is sourced from local meteorological stations and enhanced with open-meteo datasets.

Weather Map

This map shows hourly temperature across Malaysian states. Use the dropdown menus to filter by year, month, day, and hour.

You can zoom and re-center the map using the controls. The color scale indicates temperature intensity from cooler to hotter.

Weather Metrics

This line chart displays the monthly trends of precipitation, humidity, wind speed, and UV index in 2025. Hover over the lines to see detailed values and use the interactive features to analyze seasonal patterns.
